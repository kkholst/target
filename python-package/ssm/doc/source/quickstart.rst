Quickstart
====================


Installation
--------------------

Install with ``pip``:

.. code-block:: console

    pip install ssm

.. note:: Presently binary wheels are available for Linux
	  systems and Mac OS X running Python>=3.6. Windows
	  installations must be built from source.


Example
--------------------

As an illustration data is loaded from the package

.. ipython:: python
   print(2+2)
